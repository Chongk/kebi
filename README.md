`KEBI`는 핵물리 연구를 위한 프레임워크 입니다. 기본적으로 `Geant4`와 `ROOT`를 이용하고 `c++`로 쓰인 소프트웨어 패키지입니다. `Geant4`를 이용한 시뮬레이션 부터 `ROOT`를 이용한 분석 까지 공동연구에 적합한 기본 틀을 만들었습니다. `KEBI`의 가장 큰 특징은 테스크 위주의 분석을 하고 Input/Output 포멧을 통일하여 일관성을 제공한다는 점입니다. 현재 한국에서 계획하고 있는 핵물리 실험인 `LAMPS`를 중심으로 개발되고 있습니다. 이 프로젝트를 이용하기 위해서 Git(필수), ROOT(필수)와 Geant4(선택)를 다룰 줄 알아야 합니다.
